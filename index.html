Here’s the strategy in plain English:

1. Structure

  • If the market is making higher highs and higher lows, they only look for longs

  • If it is making lower highs and lower lows, they only look for shorts

  • If the market is just moving sideways, they skip trading altogether

The idea is to trade with the overall trend, not against it.

2. Level
Next, they mark important supply and demand zones on the lower time frame.

  • Demand level = area before a sharp move up

  • Supply level = area before a sharp move down

If the higher time frame is bearish, they only care about supply levels for short trades. If bullish, they would focus on demand levels for longs. They do not trade everywhere — only when price comes back to one of those pre-marked zones.

  • Enter only after confirmation

  • Ignore setups that do not meet all conditions

The strategy

Call it TPR: Trend, Pullback, Reaction

1. Trend
Use a higher time frame to decide direction.
For example:

  • Trade on the 5-minute

  • Confirm direction on the 1-hour or 4-hour

Rules:

  • If price is making higher highs and higher lows, only look for longs

  • If price is making lower highs and lower lows, only look for shorts

  • If it is choppy or sideways, do nothing

2. Pullback

Do not chase price after a big move. Wait for it to pull back into a meaningful area like:

  • VWAP

  • 9 EMA / 20 EMA area

  • Previous breakout level

  • Premarket high or low

  • Strong support/resistance zone

You want the market to “come to you.”

3. Reaction

Once price reaches that area, wait for proof that buyers or sellers are stepping in.

For a long:

  • Price pulls back to support

  • A candle rejects lower prices

  • Volume starts to pick up

  • Next candle breaks the rejection candle high

For a short:

  • Price pulls back to resistance

  • A candle rejects higher prices

  • Volume increases

  • Next candle breaks the rejection candle low

That break is the entry trigger.

4. Risk filter

This is the part most people skip.

Before entering:

  • Risk only 1% or less of your account on one trade

  • Skip trades with a bad spread

  • Skip stocks with weak volume

  • Skip random mid-chart entries

  • Do not enter unless reward is at least 2:1

Stop-loss:

  • Below the pullback low for longs

  • Above the pullback high for shorts

Take profit:

  • First target at 1R

  • Main target at 2R

  • You can move stop to breakeven after 1R if you want less downside

It combines a few strong ideas:

from AlgorithmImports import *

from collections import deque

class TPRStrategyV2(QCAlgorithm):
    """
    TPR Strategy V2
    Trend -> Pullback -> Reaction

    Improvements over v1:
    - Proper LEAN method names: Initialize / OnData
    - 5-minute execution bars via consolidator
    - 1-hour trend bars via consolidator
    - Entry trigger must occur AFTER the rejection candle
    - Real cooldown after exits
    - Bracket-style exits using stop-market + limit orders
    - ATR-based pullback distance instead of loose fixed % threshold
    """

    def Initialize(self):
        self.SetStartDate(2023, 1, 1)
        self.SetCash(100000)

        # ---------- Parameters ----------
        self.risk_per_trade = 0.01            # 1% account risk
        self.min_reward_to_risk = 2.0         # minimum 2R
        self.max_position_size = 0.25         # max 25% of portfolio in one position
        self.cooldown_minutes = 60            # no re-entry for 60 min after exit
        self.max_spread_pct = 0.002           # 0.20%
        self.pullback_atr_multiple = 0.35     # how close price must be to VWAP/EMA
        self.volume_multiplier = 1.2          # reaction candle volume threshold
        self.max_universe = 10

        tickers = ["SPY", "QQQ", "AAPL", "MSFT", "GOOGL", "AMZN", "TSLA", "NVDA", "META", "AMD"]

        self.symbol_data = {}
        self.last_exit_time = {}

        self.Settings.SeedInitialPrices = True
        self.SetBrokerageModel(BrokerageName.InteractiveBrokersBrokerage, AccountType.Margin)

        for ticker in tickers[:self.max_universe]:
            equity = self.AddEquity(ticker, Resolution.MINUTE)
            equity.SetDataNormalizationMode(DataNormalizationMode.Raw)
            symbol = equity.Symbol

            sd = SymbolData(self, symbol)
            self.symbol_data[symbol] = sd
            self.last_exit_time[symbol] = self.Time - timedelta(days=1)

        self.Debug("TPR Strategy V2 initialized")

    def OnData(self, data: Slice):
        # Main logic runs from consolidated 5-minute bars inside SymbolData.
        # We still use OnData for spread checks and safety.

    def OnOrderEvent(self, orderEvent: OrderEvent):
        if orderEvent.Status != OrderStatus.Filled:
            return

        order = self.Transactions.GetOrderById(orderEvent.OrderId)
        if order is None:
            return

        symbol = order.Symbol
        if symbol not in self.symbol_data:
            return

        sd = self.symbol_data[symbol]

        # Entry filled
        if sd.entry_order_ticket and orderEvent.OrderId == sd.entry_order_ticket.OrderId:
            sd.entry_price = orderEvent.FillPrice
            self.Debug(f"{symbol.Value}: Entry filled at {sd.entry_price:.2f}")

        # Stop filled
        if sd.stop_order_ticket and orderEvent.OrderId == sd.stop_order_ticket.OrderId:
            self.Debug(f"{symbol.Value}: Stop filled at {orderEvent.FillPrice:.2f}")
            self._handle_position_exit(symbol, "stop")

        # Target filled
        if sd.target_order_ticket and orderEvent.OrderId == sd.target_order_ticket.OrderId:
            self.Debug(f"{symbol.Value}: Target filled at {orderEvent.FillPrice:.2f}")
            self._handle_position_exit(symbol, "target")

    def _handle_position_exit(self, symbol, reason):
        sd = self.symbol_data[symbol]
        self.last_exit_time[symbol] = self.Time

        # Cancel sibling exit order if still open
        if reason == "stop" and sd.target_order_ticket is not None:
            if sd.target_order_ticket.Status not in [OrderStatus.Filled, OrderStatus.Canceled]:
                sd.target_order_ticket.Cancel("Sibling target canceled after stop fill")
        elif reason == "target" and sd.stop_order_ticket is not None:
            if sd.stop_order_ticket.Status not in [OrderStatus.Filled, OrderStatus.Canceled]:
                sd.stop_order_ticket.Cancel("Sibling stop canceled after target fill")

        sd.reset_trade_state()

    def _is_spread_acceptable(self, symbol):
        security = self.Securities[symbol]
        if security.Price <= 0:
            return False
        if security.BidPrice <= 0 or security.AskPrice <= 0:
            return False

        spread_pct = (security.AskPrice - security.BidPrice) / security.Price
        return spread_pct <= self.max_spread_pct

    def evaluate_entry(self, symbol, bar):
        """Called by SymbolData when a new 5-minute bar arrives."""
        sd = self.symbol_data[symbol]
        holdings = self.Portfolio[symbol]

        # No new entries if already invested or entry order pending
        if holdings.Invested or sd.has_open_entry_order():
            return

        # Cooldown after exit
        if self.Time < self.last_exit_time[symbol] + timedelta(minutes=self.cooldown_minutes):
            return

        # Basic spread safety
        if not self._is_spread_acceptable(symbol):
            return

        if not sd.is_ready():
            return

        # Step 1: Trend
        trend = sd.get_trend()
        if trend == TrendDirection.SIDEWAYS:
            return

        # Step 2: Pullback
        pullback_level = sd.get_pullback_level()
        if pullback_level is None:
            return

        # Step 3: Reaction must already exist from a PRIOR bar
        reaction_bar = sd.get_reaction_signal(trend)
        if reaction_bar is None:
            return

        # Step 4: Current bar must break reaction bar
        if not sd.has_entry_trigger(trend, bar):
            return

        entry_price = bar.Close
        stop_loss = sd.get_stop_loss(trend)

        if stop_loss is None:
            return

        risk_per_share = abs(entry_price - stop_loss)
        if risk_per_share <= 0:
            return

        target_price = sd.get_target_price(trend, entry_price, stop_loss)
        if target_price is None:
            return

        reward = abs(target_price - entry_price)
        risk = abs(entry_price - stop_loss)

        if risk == 0 or reward / risk < self.min_reward_to_risk:
            return

        # Position sizing
        risk_amount = self.Portfolio.TotalPortfolioValue * self.risk_per_trade
        shares = int(risk_amount / risk_per_share)

        if shares <= 0:
            return

        # Cap max notional position size
        max_position_value = self.Portfolio.TotalPortfolioValue * self.max_position_size
        max_shares_by_value = int(max_position_value / entry_price)
        shares = min(shares, max_shares_by_value)

        if shares <= 0:
            return

        quantity = shares if trend == TrendDirection.UPTREND else -shares

        # Submit entry
        sd.pending_stop_price = stop_loss
        sd.pending_target_price = target_price
        sd.pending_trend = trend

        sd.entry_order_ticket = self.MarketOrder(symbol, quantity, tag="TPR Entry")

        self.Debug(
            f"{symbol.Value}: ENTRY SIGNAL | "
            f"{'LONG' if quantity > 0 else 'SHORT'} | "
            f"Entry={entry_price:.2f} Stop={stop_loss:.2f} Target={target_price:.2f} "
            f"R:R={reward/risk:.2f} Pullback={pullback_level:.2f}"
        )

    def place_exit_orders_after_entry_fill(self, symbol):
        """Called after entry fill to place stop + target."""
        sd = self.symbol_data[symbol]
        holdings = self.Portfolio[symbol]

        if not holdings.Invested:
            return
        if sd.entry_price is None:
            return
        if sd.pending_stop_price is None or sd.pending_target_price is None:
            return

        quantity = holdings.Quantity
        exit_qty = -quantity

        # Stop-market + limit target
        sd.stop_order_ticket = self.StopMarketOrder(
            symbol,
            exit_qty,
            sd.pending_stop_price,
            tag="TPR Stop"
        )

        sd.target_order_ticket = self.LimitOrder(
            symbol,
            exit_qty,
            sd.pending_target_price,
            tag="TPR Target"
        )

        sd.stop_loss = sd.pending_stop_price
        sd.target_price = sd.pending_target_price

    def manage_open_position(self, symbol, bar):
        """Move stop to breakeven after 1R."""
        sd = self.symbol_data[symbol]
        holdings = self.Portfolio[symbol]

        if not holdings.Invested:
            return
        if sd.entry_price is None or sd.stop_loss is None or sd.target_price is None:
            return
        if sd.stop_order_ticket is None:
            return

        is_long = holdings.Quantity > 0
        risk = abs(sd.entry_price - sd.initial_stop_loss())
        if risk <= 0:
            return

        current_profit = (bar.Close - sd.entry_price) if is_long else (sd.entry_price - bar.Close)
        r_multiple = current_profit / risk

        if not sd.breakeven_moved and r_multiple >= 1.0:
            new_stop = sd.entry_price

            # Only improve the stop
            if is_long and new_stop > sd.stop_loss:
                update = UpdateOrderFields()
                update.StopPrice = new_stop
                update.Tag = "Move stop to breakeven"
                sd.stop_order_ticket.Update(update)
                sd.stop_loss = new_stop
                sd.breakeven_moved = True
                self.Debug(f"{symbol.Value}: Stop moved to breakeven at {new_stop:.2f}")

            elif not is_long and new_stop < sd.stop_loss:
                update = UpdateOrderFields()
                update.StopPrice = new_stop
                update.Tag = "Move stop to breakeven"
                sd.stop_order_ticket.Update(update)
                sd.stop_loss = new_stop
                sd.breakeven_moved = True
                self.Debug(f"{symbol.Value}: Stop moved to breakeven at {new_stop:.2f}")


class TrendDirection:
    UPTREND = 1
    DOWNTREND = -1
    SIDEWAYS = 0


class SymbolData:
    def __init__(self, algorithm: QCAlgorithm, symbol: Symbol):
        self.algorithm = algorithm
        self.symbol = symbol

        # -------- Indicators on 5-minute bars --------
        self.ema9 = ExponentialMovingAverage(9)
        self.ema20 = ExponentialMovingAverage(20)
        self.atr5 = AverageTrueRange(14, MovingAverageType.Wilders)
        self.vwap = VolumeWeightedAveragePriceIndicator(20)

        self.recent_5m_bars = deque(maxlen=30)
        self.recent_hour_bars = deque(maxlen=60)

        # Swing structure
        self.swing_highs = deque(maxlen=10)
        self.swing_lows = deque(maxlen=10)

        # Reaction tracking
        self.last_rejection_candle = None
        self.last_rejection_time = None

        # Trade state
        self.entry_order_ticket = None
        self.stop_order_ticket = None
        self.target_order_ticket = None

        self.entry_price = None
        self.stop_loss = None
        self.target_price = None
        self.pending_stop_price = None
        self.pending_target_price = None
        self.pending_trend = None
        self.breakeven_moved = False
        self._initial_stop_loss = None

        # Consolidators
        self.five_min_consolidator = TradeBarConsolidator(timedelta(minutes=5))
        self.hour_consolidator = TradeBarConsolidator(timedelta(hours=1))

        self.five_min_consolidator.DataConsolidated += self._on_five_min_bar
        self.hour_consolidator.DataConsolidated += self._on_hour_bar

        algorithm.SubscriptionManager.AddConsolidator(symbol, self.five_min_consolidator)
        algorithm.SubscriptionManager.AddConsolidator(symbol, self.hour_consolidator)

        # Warm-up indicators using consolidators
        algorithm.RegisterIndicator(symbol, self.ema9, self.five_min_consolidator)
        algorithm.RegisterIndicator(symbol, self.ema20, self.five_min_consolidator)
        algorithm.RegisterIndicator(symbol, self.atr5, self.five_min_consolidator)
        algorithm.RegisterIndicator(symbol, self.vwap, self.five_min_consolidator)

    def _on_five_min_bar(self, sender, bar: TradeBar):
        self.recent_5m_bars.append(bar)

        # If entry filled but exits not placed yet, place them
        if self.algorithm.Portfolio[self.symbol].Invested and self.entry_order_ticket is not None:
            if self.entry_order_ticket.Status == OrderStatus.Filled and self.stop_order_ticket is None:
                self._initial_stop_loss = self.pending_stop_price
                self.algorithm.place_exit_orders_after_entry_fill(self.symbol)

        # Evaluate strategy each new 5-minute bar
        if self.algorithm.Portfolio[self.symbol].Invested:
            self.algorithm.manage_open_position(self.symbol, bar)
        else:
            self.algorithm.evaluate_entry(self.symbol, bar)

    def _on_hour_bar(self, sender, bar: TradeBar):
        self.recent_hour_bars.append(bar)
        self._update_swing_points()

    def _update_swing_points(self):
        if len(self.recent_hour_bars) < 5:
            return

        bars = list(self.recent_hour_bars)[-5:]

        # center bar = bars[2]
        if (
            bars[2].High > bars[0].High and
            bars[2].High > bars[1].High and
            bars[2].High > bars[3].High and
            bars[2].High > bars[4].High
        ):
            self.swing_highs.append(bars[2].High)

        if (
            bars[2].Low < bars[0].Low and
            bars[2].Low < bars[1].Low and
            bars[2].Low < bars[3].Low and
            bars[2].Low < bars[4].Low
        ):
            self.swing_lows.append(bars[2].Low)

    def is_ready(self):
        return (
            self.ema9.IsReady and
            self.ema20.IsReady and
            self.atr5.IsReady and
            self.vwap.IsReady and
            len(self.recent_5m_bars) >= 6 and
            len(self.recent_hour_bars) >= 20 and
            len(self.swing_highs) >= 3 and
            len(self.swing_lows) >= 3
        )

    def get_trend(self):
        if len(self.swing_highs) < 3 or len(self.swing_lows) < 3:
            return TrendDirection.SIDEWAYS

        highs = list(self.swing_highs)[-3:]
        lows = list(self.swing_lows)[-3:]

        higher_highs = highs[2] > highs[1] > highs[0]
        higher_lows = lows[2] > lows[1] > lows[0]

        if higher_highs and higher_lows:
            return TrendDirection.UPTREND

        lower_highs = highs[2] < highs[1] < highs[0]
        lower_lows = lows[2] < lows[1] < lows[0]

        if lower_highs and lower_lows:
            return TrendDirection.DOWNTREND

        return TrendDirection.SIDEWAYS

    def get_pullback_level(self):
        if len(self.recent_5m_bars) == 0 or not self.atr5.IsReady:
            return None

        current_price = self.recent_5m_bars[-1].Close
        atr = self.atr5.Current.Value
        max_distance = atr * self.algorithm.pullback_atr_multiple

        candidates = [
            self.vwap.Current.Value,
            self.ema9.Current.Value,
            self.ema20.Current.Value
        ]

        for level in candidates:
            if abs(current_price - level) <= max_distance:
                return level

        return None

    def get_reaction_signal(self, trend):
        """
        Reaction must be on the PREVIOUS 5-minute bar.
        Current bar is used only as trigger bar.
        """
        if len(self.recent_5m_bars) < 2:
            return None

        reaction_bar = list(self.recent_5m_bars)[-2]

        avg_volume = sum(b.Volume for b in list(self.recent_5m_bars)[-6:-1]) / 5 if len(self.recent_5m_bars) >= 6 else reaction_bar.Volume

        body = abs(reaction_bar.Close - reaction_bar.Open)
        total_range = reaction_bar.High - reaction_bar.Low
        if total_range <= 0:
            return None

        # bullish rejection in uptrend
        if trend == TrendDirection.UPTREND:
            lower_wick = min(reaction_bar.Open, reaction_bar.Close) - reaction_bar.Low
            if lower_wick > body * 2 and reaction_bar.Volume > avg_volume * self.algorithm.volume_multiplier:
                self.last_rejection_candle = reaction_bar
                self.last_rejection_time = reaction_bar.EndTime

class SmallAccountTPR(QCAlgorithm):

    def Initialize(self):
        self.SetStartDate(2024, 1, 1)
        self.SetCash(200)

        # Use CASH account logic for small-account friendliness
        # This avoids relying on margin behavior.
        self.SetBrokerageModel(BrokerageName.InteractiveBrokersBrokerage, AccountType.Cash)

        # Parameters
        self.risk_per_trade = 0.02          # 2% risk = ~$4 on $200
        self.max_position_pct = 0.90        # can use up to 90% of account
        self.min_rr = 2.0
        self.max_spread_pct = 0.005         # 0.5% max spread
        self.pullback_atr_mult = 0.35
        self.volume_mult = 1.2

        self.min_price = 2
        self.max_price = 12
        self.max_symbols = 8
        self.max_positions = 1

        self.symbol_data = {}
        self.active_symbols = set()
        self.last_entry_date = None

        self.UniverseSettings.Resolution = Resolution.Minute
        self.AddUniverse(self.CoarseSelectionFunction)

        self.Debug("SmallAccountTPR initialized")

    def CoarseSelectionFunction(self, coarse):
        filtered = [
            c for c in coarse
            if c.HasFundamentalData
            and c.Price is not None
            and self.min_price <= c.Price <= self.max_price
            and c.Volume > 500000
            and c.DollarVolume > 5000000
        ]

        filtered = sorted(filtered, key=lambda c: c.DollarVolume, reverse=True)
        return [c.Symbol for c in filtered[:self.max_symbols]]

    def OnSecuritiesChanged(self, changes):
        for security in changes.RemovedSecurities:
            symbol = security.Symbol
            if symbol in self.symbol_data:
                self.symbol_data[symbol].Dispose()
                del self.symbol_data[symbol]
            self.active_symbols.discard(symbol)

        for security in changes.AddedSecurities:
            symbol = security.Symbol
            if symbol not in self.symbol_data:
                self.symbol_data[symbol] = SymbolState(self, symbol)
            self.active_symbols.add(symbol)

    def OnData(self, data: Slice):
        pass

    def EvaluateEntry(self, symbol, bar):
        # Only one position at a time for a $200 account
        invested_count = sum(1 for kvp in self.Portfolio.Values if kvp.Invested)
        if invested_count >= self.max_positions:
            return

        # Avoid multiple entries in one day
        if self.last_entry_date == self.Time.date():
            return

        sd = self.symbol_data[symbol]
        holdings = self.Portfolio[symbol]

        if holdings.Invested or sd.entry_ticket is not None:
            return

        # Time filter: avoid open noise, no late entries
        if self.Time.hour < 9 or (self.Time.hour == 9 and self.Time.minute < 45):
            return
        if self.Time.hour >= 14:
            return

        if not sd.IsReady():
            return

        if not self._is_spread_ok(symbol):
            return

        trend = sd.GetTrend()
        if trend != 1:   # LONG ONLY for small account
            return

        pullback = sd.GetPullbackLevel()
        if pullback is None:
            return

        reaction_bar = sd.GetReactionSignal()
        if reaction_bar is None:
            return

        if not sd.HasEntryTrigger(bar):
            return

        entry_price = bar.Close
        stop_price = sd.GetStopLoss()

        if stop_price is None or stop_price >= entry_price:
            return

        risk_per_share = entry_price - stop_price
        if risk_per_share <= 0:
            return

        target_price = entry_price + (risk_per_share * self.min_rr)

        # Risk-based size
        risk_amount = self.Portfolio.TotalPortfolioValue * self.risk_per_trade
        shares_by_risk = int(risk_amount / risk_per_share)

        # Cash cap
        max_position_value = self.Portfolio.Cash * self.max_position_pct
        shares_by_cash = int(max_position_value / entry_price)

        shares = min(shares_by_risk, shares_by_cash)

        # Must buy at least 1 share
        if shares < 1:
            return

        sd.pending_stop = stop_price
        sd.pending_target = target_price

        sd.entry_ticket = self.MarketOrder(symbol, shares, tag="SmallAccountTPR Entry")
        self.last_entry_date = self.Time.date()

        self.Debug(
            f"ENTRY {symbol.Value} | shares={shares} | entry={entry_price:.2f} "
            f"stop={stop_price:.2f} target={target_price:.2f}"
        )

    def ManageOpenPosition(self, symbol, bar):
        sd = self.symbol_data[symbol]
        holdings = self.Portfolio[symbol]

        if not holdings.Invested:
            return
        if sd.entry_fill_price is None or sd.stop_ticket is None:
            return

        initial_risk = sd.initial_stop_distance
        if initial_risk is None or initial_risk <= 0:
            return

        current_profit = bar.Close - sd.entry_fill_price
        r_multiple = current_profit / initial_risk

        # Move stop to breakeven after 1R
        if not sd.breakeven_moved and r_multiple >= 1.0:
            if sd.stop_price < sd.entry_fill_price:
                update = UpdateOrderFields()
                update.StopPrice = sd.entry_fill_price
                update.Tag = "Move stop to breakeven"
                sd.stop_ticket.Update(update)
                sd.stop_price = sd.entry_fill_price
                sd.breakeven_moved = True
                self.Debug(f"{symbol.Value}: stop moved to breakeven")

    def _is_spread_ok(self, symbol):
        sec = self.Securities[symbol]
        if sec.Price <= 0 or sec.BidPrice <= 0 or sec.AskPrice <= 0:
            return False
        spread_pct = (sec.AskPrice - sec.BidPrice) / sec.Price
        return spread_pct <= self.max_spread_pct

    def OnOrderEvent(self, orderEvent: OrderEvent):
        if orderEvent.Status != OrderStatus.Filled:
            return

        order = self.Transactions.GetOrderById(orderEvent.OrderId)
        if order is None:
            return

        symbol = order.Symbol
        if symbol not in self.symbol_data:
            return

        sd = self.symbol_data[symbol]

        # Entry fill
        if sd.entry_ticket and orderEvent.OrderId == sd.entry_ticket.OrderId:
            sd.entry_fill_price = orderEvent.FillPrice
            qty = self.Portfolio[symbol].Quantity

            if qty > 0 and sd.pending_stop is not None and sd.pending_target is not None:
                sd.stop_price = sd.pending_stop
                sd.target_price = sd.pending_target
                sd.initial_stop_distance = sd.entry_fill_price - sd.stop_price

                sd.stop_ticket = self.StopMarketOrder(symbol, -qty, sd.stop_price, tag="Stop")
                sd.target_ticket = self.LimitOrder(symbol, -qty, sd.target_price, tag="Target")

                self.Debug(
                    f"{symbol.Value}: exits placed | stop={sd.stop_price:.2f} "
                    f"target={sd.target_price:.2f}"
                )

        # Stop fill
        if sd.stop_ticket and orderEvent.OrderId == sd.stop_ticket.OrderId:
            if sd.target_ticket and sd.target_ticket.Status not in [OrderStatus.Filled, OrderStatus.Canceled]:
                sd.target_ticket.Cancel("Cancel sibling target")
            sd.ResetTradeState()
            self.Debug(f"{symbol.Value}: stopped out")

        # Target fill
        if sd.target_ticket and orderEvent.OrderId == sd.target_ticket.OrderId:
            if sd.stop_ticket and sd.stop_ticket.Status not in [OrderStatus.Filled, OrderStatus.Canceled]:
                sd.stop_ticket.Cancel("Cancel sibling stop")
            sd.ResetTradeState()
            self.Debug(f"{symbol.Value}: target hit")

        else:
            self.algo.EvaluateEntry(self.symbol, bar)

        self.hourBars.append(bar)
        self._UpdateSwings()

    def _UpdateSwings(self):
        if len(self.hourBars) < 5:
            return

        bars = list(self.hourBars)[-5:]
        mid = bars[2]

        if mid.High > bars[0].High and mid.High > bars[1].High and mid.High > bars[3].High and mid.High > bars[4].High:
            self.swingHighs.append(mid.High)

        if mid.Low < bars[0].Low and mid.Low < bars[1].Low and mid.Low < bars[3].Low and mid.Low < bars[4].Low:
            self.swingLows.append(mid.Low)

            self.ema9.IsReady and
            self.ema20.IsReady and
            self.atr.IsReady and
            self.vwap.IsReady and
            len(self.recent5) >= 6 and
            len(self.hourBars) >= 20 and
            len(self.swingHighs) >= 3 and
            len(self.swingLows) >= 3
        )

    def GetTrend(self):
        if len(self.swingHighs) < 3 or len(self.swingLows) < 3:
            return 0

        highs = list(self.swingHighs)[-3:]
        lows = list(self.swingLows)[-3:]

        up = highs[2] > highs[1] > highs[0] and lows[2] > lows[1] > lows[0]
        down = highs[2] < highs[1] < highs[0] and lows[2] < lows[1] < lows[0]

        if up:
            return 1
        if down:
            return -1
        return 0

        price = self.recent5[-1].Close

        return None

class RobinhoodTPRAlerts(QCAlgorithm):

    def Initialize(self):
        self.SetStartDate(2024, 1, 1)
        self.SetCash(200)

        # Strategy settings
        self.risk_per_trade = 0.02          # 2% risk = about $4 on a $200 account
        self.max_position_pct = 0.90        # up to 90% of account in one idea
        self.min_rr = 2.0
        self.max_spread_pct = 0.005         # 0.5%
        self.pullback_atr_mult = 0.35
        self.volume_mult = 1.2

        # Small-account universe
        self.min_price = 2
        self.max_price = 20
        self.max_symbols = 10

        # Alert controls
        self.max_alerts_per_day = 1
        self.alerts_sent_today = 0
        self.last_alert_date = None

        self.symbol_states = {}

        self.UniverseSettings.Resolution = Resolution.Minute
        self.AddUniverse(self.CoarseSelection)

        self.Debug("Robinhood TPR alert model initialized")

    def CoarseSelection(self, coarse):
        filtered = [
            c for c in coarse
            if c.HasFundamentalData
            and c.Price is not None
            and self.min_price <= c.Price <= self.max_price
            and c.Volume > 500000
            and c.DollarVolume > 5000000
        ]

        filtered = sorted(filtered, key=lambda c: c.DollarVolume, reverse=True)
        return [c.Symbol for c in filtered[:self.max_symbols]]

    def OnSecuritiesChanged(self, changes):
        for sec in changes.RemovedSecurities:
            symbol = sec.Symbol
            if symbol in self.symbol_states:
                self.symbol_states[symbol].Dispose()
                del self.symbol_states[symbol]

        for sec in changes.AddedSecurities:
            symbol = sec.Symbol
            if symbol not in self.symbol_states:
                self.symbol_states[symbol] = SymbolState(self, symbol)

    def OnData(self, data: Slice):
        # Logic handled from 5-minute consolidators
        pass

    def EvaluateSignal(self, symbol, bar):
        # Reset daily alert counter
        if self.last_alert_date != self.Time.date():
            self.last_alert_date = self.Time.date()
            self.alerts_sent_today = 0

        if self.alerts_sent_today >= self.max_alerts_per_day:
            return

        # Time window: avoid first 15 min, avoid late-day entries
        if self.Time.hour < 9 or (self.Time.hour == 9 and self.Time.minute < 45):
            return
        if self.Time.hour >= 14:
            return

        state = self.symbol_states[symbol]
        if not state.IsReady():
            return

        if not self._spread_ok(symbol):
            return

        trend = state.GetTrend()
        if trend != 1:
            return

        pullback_level = state.GetPullbackLevel()
        if pullback_level is None:
            return

        reaction_bar = state.GetReactionSignal()
        if reaction_bar is None:
            return

        if not state.HasEntryTrigger(bar):
            return

        entry = bar.Close
        stop = state.GetStopLoss()
        if stop is None or stop >= entry:
            return

        risk_per_share = entry - stop
        if risk_per_share <= 0:
            return

        target = entry + (risk_per_share * self.min_rr)
        rr = (target - entry) / (entry - stop)
        if rr < self.min_rr:
            return

        # Fractional-dollar style sizing for Robinhood manual execution
        account_value = self.Portfolio.TotalPortfolioValue
        risk_amount = account_value * self.risk_per_trade
        max_position_value = account_value * self.max_position_pct

        # Dollar amount that fits the stop-risk
        # shares = risk_amount / risk_per_share
        # dollars = shares * entry
        suggested_dollars_by_risk = (risk_amount / risk_per_share) * entry

        # Cap by max position size
        suggested_dollars = min(suggested_dollars_by_risk, max_position_value)

        # Minimum useful trade filter
        if suggested_dollars < 10:
            return

        est_fractional_shares = suggested_dollars / entry
        est_loss_at_stop = est_fractional_shares * (entry - stop)
        est_profit_at_target = est_fractional_shares * (target - entry)

        message = (
            f"BUY SETUP | {symbol.Value} | "
            f"Buy ${suggested_dollars:.2f} | "
            f"Entry {entry:.2f} | Stop {stop:.2f} | Target {target:.2f} | "
            f"RR {rr:.2f} | "
            f"Risk ${est_loss_at_stop:.2f} | "
            f"Target Profit ${est_profit_at_target:.2f}"
        )

        self.Debug(message)
        self.Log(message)

        # Optional chart annotation
        self.Plot("Signals", "Entry", entry)
        self.Plot("Signals", "Stop", stop)
        self.Plot("Signals", "Target", target)

        self.alerts_sent_today += 1

    def _spread_ok(self, symbol):
        sec = self.Securities[symbol]
        if sec.Price <= 0 or sec.BidPrice <= 0 or sec.AskPrice <= 0:
            return False

        spread_pct = (sec.AskPrice - sec.BidPrice) / sec.Price
        return spread_pct <= self.max_spread_pct


class SymbolState:
    def __init__(self, algo: QCAlgorithm, symbol: Symbol):
        self.algo = algo
        self.symbol = symbol

        self.recent5 = deque(maxlen=30)
        self.hour_bars = deque(maxlen=60)
        self.swing_highs = deque(maxlen=10)
        self.swing_lows = deque(maxlen=10)

        self.ema9 = ExponentialMovingAverage(9)
        self.ema20 = ExponentialMovingAverage(20)
        self.atr = AverageTrueRange(14, MovingAverageType.Wilders)
        self.vwap = VolumeWeightedAveragePriceIndicator(20)

        self.last_rejection = None

        self.con5 = TradeBarConsolidator(timedelta(minutes=5))
        self.con60 = TradeBarConsolidator(timedelta(hours=1))

        self.con5.DataConsolidated += self._On5Minute
        self.con60.DataConsolidated += self._OnHour

        algo.SubscriptionManager.AddConsolidator(symbol, self.con5)
        algo.SubscriptionManager.AddConsolidator(symbol, self.con60)

        algo.RegisterIndicator(symbol, self.ema9, self.con5)
        algo.RegisterIndicator(symbol, self.ema20, self.con5)
        algo.RegisterIndicator(symbol, self.atr, self.con5)
        algo.RegisterIndicator(symbol, self.vwap, self.con5)

    def _On5Minute(self, sender, bar: TradeBar):
        self.recent5.append(bar)
        self.algo.EvaluateSignal(self.symbol, bar)

    def _OnHour(self, sender, bar: TradeBar):
        self.hour_bars.append(bar)
        self._UpdateSwings()

    def _UpdateSwings(self):
        if len(self.hour_bars) < 5:
            return

        bars = list(self.hour_bars)[-5:]
        mid = bars[2]

        if mid.High > bars[0].High and mid.High > bars[1].High and mid.High > bars[3].High and mid.High > bars[4].High:
            self.swing_highs.append(mid.High)

        if mid.Low < bars[0].Low and mid.Low < bars[1].Low and mid.Low < bars[3].Low and mid.Low < bars[4].Low:
            self.swing_lows.append(mid.Low)

    def IsReady(self):
        return (
            self.ema9.IsReady and
            self.ema20.IsReady and
            self.atr.IsReady and
            self.vwap.IsReady and
            len(self.recent5) >= 6 and
            len(self.hour_bars) >= 20 and
            len(self.swing_highs) >= 3 and
            len(self.swing_lows) >= 3
        )

    def GetTrend(self):
        if len(self.swing_highs) < 3 or len(self.swing_lows) < 3:
            return 0

        highs = list(self.swing_highs)[-3:]
        lows = list(self.swing_lows)[-3:]

        up = highs[2] > highs[1] > highs[0] and lows[2] > lows[1] > lows[0]
        down = highs[2] < highs[1] < highs[0] and lows[2] < lows[1] < lows[0]

        if up:
            return 1
        if down:
            return -1
        return 0

    def GetPullbackLevel(self):
        if len(self.recent5) == 0 or not self.atr.IsReady:
            return None

        price = self.recent5[-1].Close
        max_dist = self.atr.Current.Value * self.algo.pullback_atr_mult

        for level in [self.vwap.Current.Value, self.ema9.Current.Value, self.ema20.Current.Value]:
            if abs(price - level) <= max_dist:
                return level

        return None

    def GetReactionSignal(self):
        if len(self.recent5) < 2:
            return None

        bar = list(self.recent5)[-2]
        prior = list(self.recent5)[-6:-1]
        avg_vol = sum(x.Volume for x in prior) / len(prior) if len(prior) > 0 else bar.Volume

        body = abs(bar.Close - bar.Open)
        lower_wick = min(bar.Open, bar.Close) - bar.Low
        total_range = bar.High - bar.Low

        if total_range <= 0:
            return None

        if lower_wick > body * 2 and bar.Volume > avg_vol * self.algo.volume_mult:
            self.last_rejection = bar
            return bar

        return None

    def HasEntryTrigger(self, current_bar):
        if self.last_rejection is None:
            return False
        if current_bar.EndTime <= self.last_rejection.EndTime:
            return False

        prior = list(self.recent5)[-6:-1]
        avg_vol = sum(x.Volume for x in prior) / len(prior) if len(prior) > 0 else current_bar.Volume

        return current_bar.High > self.last_rejection.High and current_bar.Volume >= avg_vol

    def GetStopLoss(self):
        if len(self.recent5) < 5:
            return None
        return min(b.Low for b in list(self.recent5)[-5:])

    def Dispose(self):
        self.algo.SubscriptionManager.RemoveConsolidator(self.symbol, self.con5)
        self.algo.SubscriptionManager.RemoveConsolidator(self.symbol, self.con60)
